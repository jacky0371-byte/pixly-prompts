# 08 细节模块（上1下2）

**版本：** v2.0-wanx
**最后更新：** 2026-03-31
**状态：** 万相迁移版
**适用模型：** wan2.6-image（图像编辑模式，enable_interleave: false）

---

## 接口参数说明

```
model: wan2.6-image
enable_interleave: false
prompt_extend: false
n: 1
size: 2K
watermark: false
```

---

## 正向提示词（text 字段）

```
图1是服装正面参考图，图2是服装工艺细节参考图。

请生成一张电商细节展示模块图，画面采用固定三格布局：上方一张半身穿着图，下方两张并排细节特写图。

【布局规格 — 严格执行】
上半区（约占总高度55%）：
- 模特穿着图1中的服装，正面上半身或半身棚拍，干净背景
- 无需人脸特写，半身构图，展示服装整体正面效果

下半区（约占总高度45%）：
- 左格（宽度50%）：图2中某处显著工艺细节的极致特写（如领口工艺、口袋细节、纽扣/拉链、袖口设计）
- 右格（宽度50%）：图2中另一处特色细节的极致特写，或面料肌理近景

各区域之间有细线或轻微间距分隔，视觉干净整洁。
每个区域内留15-20%的空白边距，供商家后期叠加文案使用。

【场景与光照】
上半区：标准棚拍光效，干净背景。
下半区：45度掠射光，展现工艺细节的立体感与精细度。
三个区域色温一致，整体视觉协调。

【服装保真 — 严格执行】
颜色为最高优先级，三个区域的服装颜色必须与图1完全一致，禁止任何区域出现色偏。
上半区服装必须与图1完全一致。
下半区细节特写必须忠实呈现图2中的真实工艺细节：
- 拉链皮环：形状、比例、材质、颜色与图2一致
- 袖口细节：罗纹/纽扣/折叠款式与图2一致
- 线迹颜色：与图2完全一致
禁止虚构或替换细节内容，选取最具代表性的工艺区域展示。

{sku_constraints_slot}

【画质要求】
三个区域均为高清锐度。
专业电商复合版式图标准。
各区域视觉权重均衡，整体布局清晰。
无水印，无文字叠加。
```

---

## 反向提示词（negative_prompt 字段）

```
模糊，噪点，低画质，变形，布局错乱，区域比例失调，细节失真，颜色偏差，文字水印
```

---

## 变量插槽说明

- `{reference_slot_1}` → content 数组第 1 个 image 字段（服装正面图）
- `{reference_slot_4}` → content 数组第 2 个 image 字段（细节图）
- `{sku_constraints_slot}` — SKU 差异约束
- ❌ 不接收补充说明

## 接口 content 数组顺序

```json
"content": [
  { "text": "（完整 prompt 文本）" },
  { "image": "{reference_slot_1}" },
  { "image": "{reference_slot_4}" }
]
```
