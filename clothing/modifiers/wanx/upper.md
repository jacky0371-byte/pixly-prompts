# 服装类型修饰器 — 上装

**版本：** v2.1-wanx

## 当前修饰器

```
主体为上装，上装占画面60%以上视觉面积。
取景从大腿中部以上，上装完整入镜。
模特面部清晰可见，光照充足。
下半身搭配简洁中性色裤子或裙子，不与上装争夺视觉注意力。
下装颜色必须与参考图一致，禁止自行替换。
镜头角度：正面平视或略俯视，突出上半身。
```
