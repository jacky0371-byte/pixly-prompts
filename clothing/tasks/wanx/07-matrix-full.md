# 07 矩阵氛围图（全身）

**版本：** v2.0-wanx
**最后更新：** 2026-03-31
**状态：** 万相迁移版
**适用模型：** wan2.6-image（图像编辑模式，enable_interleave: false）

---

## 接口参数说明

```
model: wan2.6-image
enable_interleave: false
prompt_extend: false
n: 1
size: 2K
watermark: false
```

⚠️ **风险提示**：全身多视角固定网格拼贴对万相是较高难度任务，建议实测后评估稳定性。如效果不稳定，考虑改为前端拼图方案。

---

## 正向提示词（text 字段）

```
图1是全身穿着参考图，图2是模特脸型参考图。

请生成一张电商全身矩阵氛围图：同一模特穿着同一套服装，以固定2列网格布局排列3个全身姿态在同一画面中，展现服装的动态穿着效果。

【布局构图 — 严格执行】
固定2列网格布局，禁止随机拼贴或图片重叠：
- 左列（占画面左半部分）：一张大号全身主图，正面站立，头顶到鞋底完整入镜，占满整个左列高度
- 右列（占画面右半部分）：两张等高全身图，上下排列，各占右列一半高度
  - 右上：行走动态或轻微移步姿态，全身入镜
  - 右下：四分之三全身侧面角度，全身入镜
各子图之间有一致的干净间距，每个子图有独立背景，不互相出血或重叠。
{garment_type_composition}
所有子图均为全身入镜：从头顶到鞋底。

【场景与光照】
略有电影感的氛围：都市户外感或高端室内场景。
所有子图光照温暖一致，背景柔化处理，不抢主体。
整体色调统一，有质感但不过度复杂。

【模特要求】
所有子图中的脸型、五官、配饰（眼镜等）与图2完全一致，同一张脸贯穿全部子图。
身形比例与穿搭参照图1。
姿态多样自然：站立、行走、轻微转身。
整体体态自信大方。

【服装保真 — 严格执行】
所有子图中服装颜色、结构细节、整体造型必须与图1完全一致。
颜色为最高优先级，禁止任何子图出现色偏。
全身各角度下服装长度、比例与合身度均需与图1保持一致。

{sku_constraints_slot}

【画质要求】
高清，时尚编辑摄影质感。
整体画面色调一致，视觉平衡。
无水印，无文字叠加。

{supplement_slot}
```

---

## 反向提示词（negative_prompt 字段）

```
模糊，噪点，低画质，变形，肢体扭曲，腿部变形，面部扭曲，服装颜色不一致，各子图风格不统一，图片重叠，布局混乱，文字水印
```

---

## 变量插槽说明

- `{reference_slot_6}` → content 数组第 1 个 image 字段（全身图）
- `{reference_slot_5}` → content 数组第 2 个 image 字段（模特人脸图）
- `{garment_type_composition}` — 服装类型修饰器
- `{sku_constraints_slot}` — SKU 差异约束
- `{supplement_slot}` — ✅ 接收用户补充说明

## 接口 content 数组顺序

```json
"content": [
  { "text": "（完整 prompt 文本）" },
  { "image": "{reference_slot_6}" },
  { "image": "{reference_slot_5}" }
]
```
