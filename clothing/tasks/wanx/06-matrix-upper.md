# 06 矩阵氛围图（上半身）

**版本：** v2.1-wanx
**最后更新：** 2026-04-02
**状态：** 万相调优版
**适用模型：** wan2.6-image（图像编辑模式，enable_interleave: false）

---

## 接口参数说明

```
model: wan2.6-image
enable_interleave: false
prompt_extend: false
n: 1
size: 2K
watermark: false
```

⚠️ **风险提示**：固定网格拼贴布局对万相是较高难度任务，建议实测后评估稳定性。如效果不稳定，考虑改为前端拼图方案。

---

## 正向提示词（text 字段）

```
图1是服装正面参考图，图2是模特脸型参考图。

请生成一张电商矩阵氛围图：同一模特穿着同一件服装，以固定2列网格布局排列3个上半身视角在同一画面中，营造时尚大片的编辑感。

【布局构图 — 严格执行】
固定2列网格布局，禁止随机拼贴或图片重叠：
- 左列（占画面左半部分）：一张大号上半身主图，正面或轻微侧角，占满整个左列高度
- 右列（占画面右半部分）：两张等高上半身图，上下排列，各占右列一半高度
  - 右上：四分之三侧面上半身
  - 右下：略低头或望向一侧的氛围感上半身
各子图之间有一致的干净间距，每个子图有独立背景，不互相出血或重叠。
{garment_type_composition}
所有子图均以上半身为主要展示区域。

【场景与光照】
比标准棚拍略有氛围感：暖调柔光，如高端展厅自然光或室内漫射光。
所有子图光照色调一致，整体画面氛围统一。
背景为略深于纯白的暖灰色调。

【模特要求】
所有子图中的脸型、五官、配饰（眼镜等）必须与图2完全一致，同一张脸贯穿全部子图。
各子图姿态自然变化：一张自信、一张放松、一张轻微侧转。
表情有时尚感，非硬照式僵硬。

【服装保真 — 严格执行】
所有子图中服装颜色、结构细节、合身度必须与图1完全一致。
颜色为最高优先级，禁止任何子图出现色偏。

{sku_constraints_slot}

【画质要求】
高清，时尚编辑摄影质感。
整体画面色调统一，视觉平衡。
无水印，无文字叠加。

{supplement_slot}
```

---

## 反向提示词（negative_prompt 字段）

```
模糊，噪点，低画质，变形，肢体扭曲，面部扭曲，服装颜色不一致，各子图风格不统一，图片重叠，布局混乱，文字水印
```

---

## 变量插槽说明

- `{reference_slot_1}` → content 数组第 1 个 image 字段（服装正面图）
- `{reference_slot_5}` → content 数组第 2 个 image 字段（模特人脸图）
- `{garment_type_composition}` — 服装类型修饰器
- `{sku_constraints_slot}` — SKU 差异约束
- `{supplement_slot}` — ✅ 接收用户补充说明

## 接口 content 数组顺序

```json
"content": [
  { "text": "（完整 prompt 文本）" },
  { "image": "{reference_slot_1}" },
  { "image": "{reference_slot_5}" }
]
```
