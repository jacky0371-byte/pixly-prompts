# 03 4/5轻侧展示

**版本：** v2.1-wanx
**最后更新：** 2026-04-02
**状态：** 万相调优版
**适用模型：** wan2.6-image（图像编辑模式，enable_interleave: false）

---

## 接口参数说明

```
model: wan2.6-image
enable_interleave: false
prompt_extend: false
n: 1
size: 2K
watermark: false
```

---

## 正向提示词（text 字段）

```
参考图说明：图1为服装正面实物照，图2为模特脸部参考。生成图中的服装必须严格还原图1的外观，从四分之三侧面角度展示。

电商服装侧面展示图，专业棚拍，四分之三侧面视角。

模特身体向侧面转动约30-45度，面部再向镜头方向多转约20度，形成身侧脸朝前的自然站姿。脸型五官与图2一致，表情自然，眼神朝向镜头。前脚略前迈，重心自然转移。手臂不遮挡侧面口袋与侧缝。

{garment_type_composition}

背景：纯色浅米色或米白色，与整套图风格一致。
光照：柔和方向性光，面向镜头侧略强，制造轻微立体层次。服装三维形态与垂感清晰展现。

【服装核心特征 — 必须精确还原图1】
{sku_constraints_slot}

【侧面角度特别注意】
侧缝线位置与结构清晰还原。
侧袋（如有）在此角度的可见度与位置准确。
袖型侧面垂落形态自然。
服装在侧面角度的合身度与包裹感与图1一致。

【通用保真规则】
服装颜色、面料质感、光泽度严格还原图1，禁止偏色。
所有可见结构元素的位置、形状、材质、颜色与图1一致。
禁止简化侧面结构细节。

画质：超高清，商业摄影级锐度，与正面背面风格统一。无水印无文字。

{supplement_slot}
```

## 反向提示词（negative_prompt 字段）

```
模糊，噪点，低画质，变形，肢体扭曲，面部畸变，服装结构错误，侧缝位置错误，拼接位置错误，颜色偏差，色调偏绿，色调偏灰，材质错误，皮革变布料，细节丢失，AI感，蜡像感，过度磨皮，背景杂乱，文字水印
```

---

## 变量插槽说明

- `{reference_slot_1}` → content 数组第 1 个 image 字段（服装正面图）
- `{reference_slot_5}` → content 数组第 2 个 image 字段（模特人脸图）
- `{garment_type_composition}` — 服装类型修饰器
- `{sku_constraints_slot}` — ⚠️ **SKU 差异约束（v2.1起为必填核心段）**
- `{supplement_slot}` — ✅ 接收用户补充说明

## 接口 content 数组顺序

```json
"content": [
  { "image": "{reference_slot_1}" },
  { "image": "{reference_slot_5}" },
  { "text": "（完整 prompt 文本）" }
]
```
