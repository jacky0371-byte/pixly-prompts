# 01 正面主展示

**版本：** v2.1-wanx
**最后更新：** 2026-04-02
**状态：** 万相调优版
**适用模型：** wan2.6-image（图像编辑模式，enable_interleave: false）

---

## 接口参数说明

```
model: wan2.6-image
enable_interleave: false
prompt_extend: false   ← 关闭自动润色，保证结构约束精确执行
n: 1
size: 2K
watermark: false
```

---

## 正向提示词（text 字段）

```
参考图说明：图1为服装正面实物照，图2为模特脸部参考。生成图中的服装必须严格还原图1的外观。

电商服装主图，专业棚拍，正面视角。

模特正面站立，居中，直面镜头。脸型五官与图2一致，表情自然自信，眼神看向镜头。站姿放松，双臂自然下垂。发型整洁不遮挡领口肩部。

{garment_type_composition}

背景：纯色浅米色或米白色，干净无杂物。
光照：专业柔光棚拍，正面左侧主光，右侧补光，服装无硬阴影，面料质感清晰。脚下轻微自然阴影。

【服装核心特征 — 必须精确还原图1】
{sku_constraints_slot}

【通用保真规则】
服装颜色、面料质感、光泽度严格还原图1，禁止偏色、提亮或美化。
所有结构元素（领型、袖口、口袋、拉链、纽扣、拼接、缝线、印花）的位置、形状、材质、颜色必须与图1一致。
禁止遗漏、简化、添加或移动任何设计元素。
禁止用"常见款式"替代图1的实际设计。

画质：超高清，商业摄影级锐度，无水印无文字无边框。

{supplement_slot}
```

## 反向提示词（negative_prompt 字段）

```
模糊，噪点，低画质，变形，肢体扭曲，多余手指，面部畸变，服装结构错误，口袋位置偏移，纽扣数量错误，拉链位置错误，颜色偏差，色调偏绿，色调偏灰，材质错误，皮革变布料，拼接位置错误，拼接形状错误，细节丢失，缝线消失，AI感，蜡像感，塑料感，过度磨皮，过度锐化，背景杂乱，背景有物体，文字，水印，边框，logo
```

---

## 变量插槽说明

- `{reference_slot_1}` → content 数组第 1 个 image 字段（服装正面图）
- `{reference_slot_5}` → content 数组第 2 个 image 字段（模特人脸图）
- `{garment_type_composition}` — 服装类型修饰器注入位置
- `{sku_constraints_slot}` — ⚠️ **SKU 差异约束注入位置（v2.1起为必填核心段，非可选补充）**
- `{supplement_slot}` — ✅ 接收用户补充说明

## 接口 content 数组顺序

```json
"content": [
  { "image": "{reference_slot_1}" },
  { "image": "{reference_slot_5}" },
  { "text": "（完整 prompt 文本）" }
]
```

> ⚠️ **v2.1 变更：图片放在 text 之前。** 万相在 content 数组中对靠前的元素给予更高权重，图片前置有助于提升参考图的影响力。
