# 10 吊牌展示图

**版本：** v2.1-wanx
**最后更新：** 2026-04-02
**状态：** 万相调优版
**适用模型：** wan2.6-image（图像编辑模式，enable_interleave: false）

---

## 接口参数说明

```
model: wan2.6-image
enable_interleave: false
prompt_extend: false
n: 1
size: 2K
watermark: false
```

⚠️ **风险提示**：吊牌文字精确还原是万相的高风险任务，模型倾向于"模仿文字外形"而非精确复制内容。建议实测后评估文字清晰度，如不满足要求可考虑前端叠字方案替代。

---

## 正向提示词（text 字段）

```
图1是吊牌正面参考图，图2是吊牌反面（洗标/成分标）参考图。

请生成一张清晰干净的吊牌展示图：上方展示吊牌正面，下方展示吊牌反面，所有文字、标志、符号必须清晰可读。

【构图】
两段式布局：
- 上半区：图1吊牌正面特写，展示品牌名、logo及主要信息，填满上半区
- 下半区：图2吊牌反面特写，展示面料成分、尺码、洗涤说明，填满下半区
两张吊牌均为特写构图，充满各自区域。
吊牌可略有角度（非完全正视），可置于轻微织物背景上增加质感。
两区之间有干净分隔。

【光照】
均匀明亮的正面光照：文字区域无阴影遮挡。
无高光反射（尤其是塑料或光面吊牌）。
柔和中性背景光。

【文字与内容保真 — 严格执行】
吊牌上所有内容必须与参考图严格一致：
- 所有文字、数字、字母的内容与排版
- 品牌标志的形状与细节
- 洗涤符号的正确形态（禁止生成错误的洗涤图标）
- 吊牌底色、印刷色、logo颜色与参考图完全一致，禁止美化或偏色
禁止生成虚构的品牌名称或虚构的洗涤信息。
禁止用通用吊牌模板替代参考图内容。

{sku_constraints_slot}

【画质要求】
文字区域超高清锐度，标准阅读尺寸下完全可读。
专业商品细节摄影。
干净可信赖的整体呈现。
无水印，无额外文字叠加。
```

---

## 反向提示词（negative_prompt 字段）

```
模糊，噪点，低画质，文字模糊，文字变形，错误洗涤符号，虚构品牌信息，阴影遮挡文字，反光，颜色偏差，AI感，文字水印
```

---

## 变量插槽说明

- `{reference_slot_7}` → content 数组第 1 个 image 字段（吊牌正面）
- `{reference_slot_8}` → content 数组第 2 个 image 字段（吊牌反面）
- `{sku_constraints_slot}` — SKU 差异约束
- ❌ 不接收补充说明

## 接口 content 数组顺序

```json
"content": [
  { "text": "（完整 prompt 文本）" },
  { "image": "{reference_slot_7}" },
  { "image": "{reference_slot_8}" }
]
```
