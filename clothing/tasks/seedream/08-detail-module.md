extra_body={

## Prompt 文本

```
图1是服装正面参考图，图2是服装工艺细节参考图。

请生成一张电商细节展示模块图，画面采用固定三格布局：上方一张半身穿着图，下方两张并排细节特写图。

布局：
上半区（约占总高度55%）：模特穿着图1中的服装，正面上半身棚拍，干净背景。
下半区（约占总高度45%）：左格为图2中某处显著工艺细节的极致特写（如领口工艺、口袋细节、纽扣/拉链、袖口设计），右格为另一处特色细节的极致特写或面料肌理近景。
各区域之间有细线或轻微间距分隔。每个区域内留15-20%空白边距，供后期叠加文案。

光照：上半区标准棚拍光效。下半区45度掠射光展现工艺立体感。三个区域色温一致。

服装保真：三个区域的服装颜色必须与图1完全一致。上半区服装与图1一致。下半区细节特写忠实呈现图2中的真实工艺（拉链皮环形状、袖口细节、线迹颜色等），不得虚构或替换。

{sku_constraints_slot}

画质：三个区域均为高清锐度。专业电商复合版式图。无水印无文字。
```

## 变量插槽说明

- `{reference_slot_1}` → extra_body.image[0]
- `{reference_slot_4}` → extra_body.image[1]
- `{sku_constraints_slot}` — SKU 差异约束
