# 02 背面展示

**版本：** v2.0-wanx
**最后更新：** 2026-03-31
**状态：** 万相迁移版
**适用模型：** wan2.6-image（图像编辑模式，enable_interleave: false）

---

## 接口参数说明

```
model: wan2.6-image
enable_interleave: false
prompt_extend: false
n: 1
size: 2K
watermark: false
```

---

## 正向提示词（text 字段）

```
图1是服装背面穿着参考图，图2是模特脸型参考图。

请生成一张电商背面展示图：模特背对镜头穿着图1中的服装，展示服装背面全貌。

【构图】
{garment_type_composition}
模特背对镜头居中站立，头部向左或向右轻微转动约15-20度，隐约可见侧脸轮廓，增加画面生动感，但不喧宾夺主。

【场景与光照】
干净极简背景，与正面主图风格保持一致（浅米色或米白色）。
柔和均匀的棚拍光效，充分照亮背部细节。
背部缝线、口袋、拉链、结构细节清晰可见，无遮挡阴影。

【模特要求】
侧脸轮廓与图2保持一致。
挺拔自信的站姿，肩膀打开，脊背自然伸展。
双臂自然垂落，不遮挡服装背部任何细节。
发型拨向一侧，确保后领口与领座完整可见。

【服装保真 — 严格执行】
服装背面每一处结构细节必须与图1完全一致，包括：
- 背部面板拼接、缝线、省道
- 背部口袋、拉链、开衩、开叉
- 领口与领座从背面看的形状
- 下摆形状与长度
- 背部印花、织标、刺绣等装饰元素
禁止添加或改变图1中不存在的任何背部结构元素。

{sku_constraints_slot}

【画质要求】
超高清，商业摄影锐度。
与正面主图光照风格保持一致。
无水印，无文字叠加。
```

---

## 反向提示词（negative_prompt 字段）

```
模糊，噪点，低画质，变形，肢体扭曲，服装背部结构变形，背部细节缺失，颜色偏差，背景杂乱，文字水印
```

---

## 变量插槽说明

- `{reference_slot_2}` → content 数组第 1 个 image 字段（服装背面穿着图）
- `{reference_slot_5}` → content 数组第 2 个 image 字段（模特人脸图）
- `{garment_type_composition}` — 服装类型修饰器
- `{sku_constraints_slot}` — SKU 差异约束
- ❌ 不接收补充说明

## 接口 content 数组顺序

```json
"content": [
  { "text": "（完整 prompt 文本）" },
  { "image": "{reference_slot_2}" },
  { "image": "{reference_slot_5}" }
]
```
