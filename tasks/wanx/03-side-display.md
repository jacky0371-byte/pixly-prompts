# 03 4/5轻侧展示

**版本：** v2.0-wanx
**最后更新：** 2026-03-31
**状态：** 万相迁移版
**适用模型：** wan2.6-image（图像编辑模式，enable_interleave: false）

---

## 接口参数说明

```
model: wan2.6-image
enable_interleave: false
prompt_extend: false
n: 1
size: 2K
watermark: false
```

---

## 正向提示词（text 字段）

```
图1是服装正面参考图，图2是模特脸型参考图。

请生成一张电商四分之三侧面展示图：模特身体向侧面转动约30-45度（4/5侧面角度），面部略向镜头方向转，展示服装的立体轮廓与侧面构造。

【构图】
{garment_type_composition}
模特身体转动30-45度，形成自然的四分之三角度。
面部相对身体再向镜头方向多转约20度，实现"身侧面、脸四分之三"的经典站姿。
前侧脚略向前迈出，形成自然重心转移，站姿生动。

【场景与光照】
干净极简背景，与整套图风格一致。
柔和方向性光照，主光来自面向镜头的一侧，制造轻微立体感与层次感。
光照充分展现服装的三维形态与面料垂感，不产生遮挡细节的阴影。

【模特要求】
脸型、五官与图2严格一致。
表情自然，眼神朝向镜头或略过镜头方向。
手臂位置自然，不遮挡侧面口袋、侧缝等关键结构细节。

【服装保真 — 严格执行】
服装所有结构细节必须与图1完全一致。
从侧面角度需特别还原：
- 侧缝线位置与结构
- 侧袋可见度与位置（如有）
- 袖型从侧面的垂落形态
- 服装在这一角度下包裹身体的方式与合身度
- 非对称设计元素在此角度的呈现
禁止简化侧面结构细节。

{sku_constraints_slot}

【画质要求】
超高清，商业摄影锐度。
与正面主图和背面图风格统一。
无水印，无文字叠加。

{supplement_slot}
```

---

## 反向提示词（negative_prompt 字段）

```
模糊，噪点，低画质，变形，肢体扭曲，服装结构变形，侧缝错误，颜色偏差，背景杂乱，文字水印
```

---

## 变量插槽说明

- `{reference_slot_1}` → content 数组第 1 个 image 字段（服装正面图）
- `{reference_slot_5}` → content 数组第 2 个 image 字段（模特人脸图）
- `{garment_type_composition}` — 服装类型修饰器
- `{sku_constraints_slot}` — SKU 差异约束
- `{supplement_slot}` — ✅ 接收用户补充说明

## 接口 content 数组顺序

```json
"content": [
  { "text": "（完整 prompt 文本）" },
  { "image": "{reference_slot_1}" },
  { "image": "{reference_slot_5}" }
]
```
