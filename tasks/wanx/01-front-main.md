# 01 正面主展示

**版本：** v2.0-wanx
**最后更新：** 2026-03-31
**状态：** 万相迁移版
**适用模型：** wan2.6-image（图像编辑模式，enable_interleave: false）

---

## 接口参数说明

```
model: wan2.6-image
enable_interleave: false
prompt_extend: false   ← 关闭自动润色，保证结构约束精确执行
n: 1
size: 2K
watermark: false
```

---

## 正向提示词（text 字段）

```
图1是服装正面参考图，图2是模特脸型参考图。

请生成一张电商主图：模特正面站立穿着图1中的服装，脸型与图2保持一致。

【构图】
{garment_type_composition}
模特居中站立，直面镜头，服装为画面视觉焦点。

【场景与光照】
干净极简背景，浅米色、米白色或浅暖灰色纯色背景。
专业棚拍光效：主光从正面左侧柔光打入，右侧补光，服装正面无硬阴影。
服装区域光照均匀，充分展现面料质感与工艺细节。
模特脚底有自然轻微落地阴影。

【模特要求】
脸型、五官、脸部轮廓与图2严格一致。
表情自然自信，轻微微笑或中性表情。
站姿放松，双臂自然垂落或一手轻搭服装。
眼神看向镜头或略偏镜头方向。
发型整洁，不遮挡服装领口与肩部。

【服装保真 — 严格执行】
服装每一处结构细节必须与图1完全一致，包括：
- 领口形状与深度
- 袖长、袖宽、袖口设计
- 口袋：位置、大小、形状、袋盖设计，完全按图1还原
- 纽扣、拉链、按扣：位置与款式完全一致
- 缝线、明线、装饰车线
- 面料质感、光泽、垂感、厚度感
- 颜色：严格还原图1色调，禁止偏色或美化
- 印花、图案、logo、刺绣：位置与比例完全一致
禁止简化、添加、删除或移动任何结构元素。
禁止按"常见款式"默认生成，必须忠实还原图1。

{sku_constraints_slot}

【画质要求】
超高清，商业摄影锐度。
无水印，无文字叠加，无边框。

{supplement_slot}
```

## 反向提示词（negative_prompt 字段）

```
模糊，噪点，低画质，变形，肢体扭曲，多余手指，面部扭曲，服装结构变形，口袋位置错误，颜色偏差，AI感，蜡像感，过度磨皮，背景杂乱，文字水印，边框
```

---

## 变量插槽说明

- `{reference_slot_1}` → content 数组第 1 个 image 字段（服装正面图）
- `{reference_slot_5}` → content 数组第 2 个 image 字段（模特人脸图）
- `{garment_type_composition}` — 服装类型修饰器注入位置
- `{sku_constraints_slot}` — SKU 差异约束注入位置
- `{supplement_slot}` — ✅ 接收用户补充说明

## 接口 content 数组顺序

```json
"content": [
  { "text": "（完整 prompt 文本）" },
  { "image": "{reference_slot_1}" },
  { "image": "{reference_slot_5}" }
]
```
