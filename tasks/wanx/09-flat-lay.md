# 09 平铺正反面

**版本：** v2.0-wanx
**最后更新：** 2026-03-31
**状态：** 万相迁移版
**适用模型：** wan2.6-image（图像编辑模式，enable_interleave: false）

---

## 接口参数说明

```
model: wan2.6-image
enable_interleave: false
prompt_extend: false
n: 1
size: 2K
watermark: false
```

---

## 正向提示词（text 字段）

```
图1是服装正面参考图，图2是服装平铺背面参考图。

请生成一张无模特纯商品平铺图：上方展示服装正面平铺，下方展示服装背面平铺，正反面对齐排列，方便消费者直观对比。

【构图】
上半区：服装正面平铺，俯拍视角（正上方垂直向下）
下半区：服装背面平铺，俯拍视角，与正面保持相同比例和对齐
两件服装的宽度、比例完全一致，垂直居中对齐。
服装摆放整洁：无褶皱，袖子/裤腿对称摆放。
两区之间留适当间距，干净分隔。

【场景与光照】
纯色平铺台面：白色、米白色或极浅灰色。
完全均匀的正上方平行光：无方向性阴影，无渐变，无景深效果。
平铺光照必须保持服装真实颜色，不因过曝导致深色偏浅，不因阴影导致局部偏暗。

【服装保真 — 严格执行】
颜色为最高优先级，正面和背面均需精确还原参考图颜色，禁止偏色。
正面平铺必须与图1完全一致：
- 所有口袋、纽扣、拉链的精确位置
- 所有缝线、明线、印花/图案
- 精确颜色与面料质感
背面平铺必须与图2完全一致：
- 背部面板结构、开衩、口袋、织标
- 所有背部结构细节完整呈现
正反面明确为同一件服装，视觉风格完全统一。

{sku_constraints_slot}

【画质要求】
超高清，曝光均匀。
专业商品平铺摄影标准。
无褶皱，无倾斜，服装摆放完美。
无水印，无文字叠加。
```

---

## 反向提示词（negative_prompt 字段）

```
模糊，噪点，低画质，褶皱，倾斜，方向性阴影，渐变背景，景深，服装结构变形，颜色偏差，文字水印
```

---

## 变量插槽说明

- `{reference_slot_1}` → content 数组第 1 个 image 字段（正面图）
- `{reference_slot_3}` → content 数组第 2 个 image 字段（平铺背面图）
- `{sku_constraints_slot}` — SKU 差异约束
- ❌ 不接收补充说明
- 无需人脸参考图

## 接口 content 数组顺序

```json
"content": [
  { "text": "（完整 prompt 文本）" },
  { "image": "{reference_slot_1}" },
  { "image": "{reference_slot_3}" }
]
```
