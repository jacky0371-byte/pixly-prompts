# 05 肌理细节图

**版本：** v2.0-wanx
**最后更新：** 2026-03-31
**状态：** 万相迁移版
**适用模型：** wan2.6-image（图像编辑模式，enable_interleave: false）

---

## 接口参数说明

```
model: wan2.6-image
enable_interleave: false
prompt_extend: false
n: 1
size: 2K
watermark: false
```

---

## 正向提示词（text 字段）

```
图1（至图4）是同一件服装的细节/肌理参考图，可能包含面料特写、工艺细节、缝线、配件等多个角度。

请生成一张极致特写的面料肌理展示图，让观看者通过图片感受到材质的真实质感。

【构图】
超近距离微距特写构图，面料细节占画面80%以上。
不展示完整服装轮廓，只聚焦面料局部。
可以包含一条缝线、一粒纽扣边缘或一处拉链头作为比例参照点。
画面略有角度倾斜（非完全平铺），展现面料的立体感和垂感。

【光照】
45度侧向掠射光，充分展现织物纹理深度与立体感。
高对比但柔和，每一根纱线、每一处编织结构清晰可见。
高光区域不过曝，保留最亮处的面料细节。

【面料保真 — 严格执行】
面料质感必须与参考图严格一致，包括：
- 织法结构（平纹、斜纹、针织、罗纹等）完整还原
- 纤维外观（棉的哑光感、丝的光泽、羊毛的绒感等）忠实呈现
- 近距离下的颜色准确度，禁止偏色或美化
- 可见的缝线质量与线色，线色必须与参考图完全一致
- 任何表面处理工艺（水洗效果、做旧处理、涂层等）
禁止用通用面料质感替代，必须忠实还原参考图中的真实材质。

{sku_constraints_slot}

【画质要求】
微距级锐度，每一根纱线清晰可辨。
专业纺织品摄影标准。
无水印，无文字叠加。
```

---

## 反向提示词（negative_prompt 字段）

```
模糊，噪点，低画质，面料质感失真，通用面料替代，过度磨皮，颜色偏差，文字水印
```

---

## 变量插槽说明

- `{reference_slot_4}` → content 数组第 1-4 个 image 字段（细节图，最多 4 张）
- `{sku_constraints_slot}` — SKU 差异约束
- ❌ 不接收补充说明
- 无需人脸参考图

## 接口 content 数组顺序

```json
"content": [
  { "text": "（完整 prompt 文本）" },
  { "image": "{reference_slot_4[0]}" },
  { "image": "{reference_slot_4[1]}" }
]
```

注：有几张细节图传几张，最多 4 张。
